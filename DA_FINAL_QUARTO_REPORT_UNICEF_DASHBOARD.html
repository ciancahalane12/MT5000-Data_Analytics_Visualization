Preface¶

This dashboard presents world data from UNICEF database of the following indicators across countries both Male and Female Gender:
  1. Total Population (2011-2023)
  2. Life Expectancy at birth (2011-2023)
  3. Moderate Homogeneous Standards, At least TWO Deprivations in Children (2011-2023)

What the Dashboard is about?

Key demographic variables like total population, life expectancy, and child deprivation are visualised in the dashboard. key visualisations make links, patterns, and trends between key indicators across time and across nations apparent. Users can explore particular nations using the interactive features and learn more about how things like the Covid-19 outbreak affect life expectancy.

Data Sources¶

The dashboard uses data from UNICEF data files:

  • Total_Population.csv
  • Life_expectancy.csv
  • Deprivation_Children.csv
